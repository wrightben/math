f'(x) = 0 yields critical numbers (c); Note that one of the critical numbers of the graphed equation falls on x = 0, which means the mouse has to be on the y-axis to detect the value, because the axis and the graphed x = 0 are indistinguishable.

f(x) is concave up (smile) when \( f''( f'(x) = 0 ) > 0 \); f(x) is concave down (frown) when \( f''( f'(x) = 0 ) < 0 \).
