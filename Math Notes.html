First and Second Derivative Test.gcx ( Critical Numbers and Concavity )

\(f'(x) = 0\) yields critical numbers.

Note that if one of the critical numbers of the graphed equation falls on x = 0, the mouse has to be on the y-axis to detect the value, because the axis and the graphed x = 0 are indistinguishable.

f(x) is concave up (smile) when \( f''( f'(x) = 0 ) > 0 \); f(x) is concave down (frown) when \( f''( f'(x) = 0 ) < 0 \).
